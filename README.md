# Rxjs-Playground

A Rxjs - Typescript - Nodemon environment.  
Auto refresh on save, compile the typescript and ready to use for rxjs.


## Installation

Use [npm](https://www.npmjs.com/get-npm) to install the dependencies
```bash
npm install
```

## Usage
run:
```bash
npm run-script play
```
